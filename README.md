# GC_Skew_Project
with Yoonho Choi, starting from 2023<br>
<br>
GC skew = (G - C) / (G + C)<br>
GC Difference = (G - C)<br>
<br>
You can download FASTA files at NCBI (genome recommended)<br>
<br>
The recommended datasets for this analysis include<br>
Archaea, Bacteria, Mitochondria, and Chloroplast genomes. <br>
Although general eukaryotic sequences are compatible, they often fail to demonstrate significant patterns.
